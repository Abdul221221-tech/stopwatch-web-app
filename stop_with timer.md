⏱️ Stopwatch Web Application (HTML, CSS, JavaScript)

A simple, responsive, and interactive Stopwatch web application built using vanilla HTML, CSS, and JavaScript.
This project demonstrates core JavaScript concepts like DOM manipulation, events, and timing functions (setInterval, clearInterval).

The stopwatch includes essential features like Start, Stop, Reset, Get Time (Lap), and Clear functionality with a modern UI and responsive design.

✨ Features
▶️ Start / Stop timer functionality
🔄 Reset stopwatch to zero
📌 Lap feature (store multiple time records)
🧹 Clear all recorded lap times
📱 Fully responsive design (mobile + desktop friendly)
🎨 Modern UI with gradient background and smooth buttons
⚡ Lightweight and fast (no external libraries)
🛠️ Tech Stack
HTML5
CSS3 (Flexbox + Responsive Design)
JavaScript (DOM + Events + Timing Functions)

🎯 How It Works
Start Button → Starts timer using setInterval()
Stop Button → Stops timer using clearInterval()
Reset Button → Resets count to 0
Get Time Button → Stores current time as a lap
Clear Button → Removes all saved lap times
